# TOOLS.md - Notes sur les outils utilisateur (modifiables)

Ce fichier est destiné à _vos_ notes sur les outils externes et les conventions.
Il ne définit pas les outils existants ; OpenClaw fournit des outils intégrés en interne.

## Exemples

### imsg

- Envoyer un iMessage/SMS : décrire qui/quoi, confirmer avant l'envoi.
- Préférez les messages courts ; évitez d'envoyer des secrets.

### sag

- Synthèse vocale : spécifier la voix, le haut-parleur/la pièce cible et s'il faut diffuser en continu.

Ajoutez tout ce que vous souhaitez que l'assistant sache sur votre chaîne d'outils locale.
