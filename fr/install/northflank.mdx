---
title: Déployer sur Northflank
---

Déployez OpenClaw sur Northflank avec un modèle en un clic et termine la configuration dans votre navigateur.
C'est le chemin le plus simple « sans terminal sur le serveur » : Northflank exécute la Gateway pour vous,
et vous configurez tout via l'assistant web `/setup`.

## Comment démarrer

1. Cliquez sur [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) pour ouvrir le modèle.
2. Créez un [compte sur Northflank](https://app.northflank.com/signup) si vous n'en avez pas déjà un.
3. Cliquez sur **Deploy OpenClaw now**.
4. Définissez la variable d'environnement requise : `SETUP_PASSWORD`.
5. Cliquez sur **Deploy stack** pour construire et exécuter le modèle OpenClaw.
6. Attendez que le déploiement soit terminé, puis cliquez sur **View resources**.
7. Ouvrez le service OpenClaw.
8. Ouvrez l'URL publique OpenClaw et terminez la configuration sur `/setup`.
9. Ouvrez l'interface de contrôle sur `/openclaw`.

## Ce que vous obtenez

- OpenClaw Gateway hébergé + Interface de contrôle
- Configuration Web sur `/setup` (aucune commande de terminal)
- Stockage persistant via Northflank Volume (`/data`) pour que la configuration/les identifiants/l'espace de travail survivent aux redéploiements

## Flux de configuration

1. Visitez `https://<your-northflank-domain>/setup` et entrez votre `SETUP_PASSWORD`.
2. Choisissez un fournisseur de modèle/d'authentification et collez votre clé.
3. (Facultatif) Ajoutez les jetons Telegram/Discord/Slack.
4. Cliquez sur **Run setup**.
5. Ouvrez l'interface de contrôle sur `https://<your-northflank-domain>/openclaw`

Si les DMs Telegram sont configurés pour l'appairage, la configuration Web peut approuver le code d'appairage.

## Obtenir les jetons de chat

### Jeton de bot Telegram

1. Envoyez un message à `@BotFather` sur Telegram
2. Exécutez `/newbot`
3. Copiez le jeton (ressemble à `123456789:AA...`)
4. Collez-le dans `/setup`

### Jeton de bot Discord

1. Allez sur [https://discord.com/developers/applications](https://discord.com/developers/applications)
2. **New Application** → choisissez un nom
3. **Bot** → **Add Bot**
4. **Enable MESSAGE CONTENT INTENT** sous Bot → Privilèges des Intents de Gateway (requis sinon le bot plantera au démarrage)
5. Copiez le **Bot Token** et collez-le dans `/setup`
6. Invitez le bot sur votre serveur (OAuth2 URL Generator ; portées : `bot`, `applications.commands`)
