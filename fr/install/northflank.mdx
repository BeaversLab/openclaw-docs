Déployez OpenClaw sur Northflank avec un modèle en un clic et terminez la configuration dans votre navigateur.
C'est le moyen le plus simple « sans terminal sur le serveur » : Northflank exécute le Gateway pour vous,
et vous configurez tout via l'assistant `/setup` web.

## Comment démarrer

1. Cliquez sur [Déployer OpenClaw](https://northflank.com/stacks/deploy-openclaw) pour ouvrir le modèle.
2. Créez un [compte sur Northflank](https://app.northflank.com/signup) si vous n'en avez pas déjà un.
3. Cliquez sur **Déployer OpenClaw maintenant**.
4. Définissez la variable d'environnement requise : `SETUP_PASSWORD`.
5. Cliquez sur **Déployer la stack** pour construire et exécuter le modèle OpenClaw.
6. Attendez que le déploiement soit terminé, puis cliquez sur **Voir les ressources**.
7. Ouvrez le service OpenClaw.
8. Ouvrez l'URL publique OpenClaw et terminez la configuration à `/setup`.
9. Ouvrez l'interface de contrôle à `/openclaw`.

## Ce que vous obtenez

- Gateway OpenClaw hébergé + Interface de contrôle
- Configuration Web à `/setup` (pas de commandes terminal)
- Stockage persistant via Northflank Volume (`/data`) pour que la configuration/les identifiants/l'espace de travail survivent aux redéploiements

## Flux de configuration

1. Visitez `https://<your-northflank-domain>/setup` et entrez votre `SETUP_PASSWORD`.
2. Choisissez un model/fournisseur d'auth et collez votre clé.
3. (Facultatif) Ajoutez les jetons Telegram/Discord/Slack.
4. Cliquez sur **Exécuter la configuration**.
5. Ouvrez l'interface de contrôle à `https://<your-northflank-domain>/openclaw`

Si les DMs Telegram sont définis sur l'appairage, la configuration Web peut approuver le code d'appairage.

## Connecter un channel

Collez votre jeton Telegram ou Discord dans l'assistant `/setup`. Pour les instructions
de configuration, consultez la documentation du channel :

- [Telegram](/fr/channels/telegram) (le plus rapide — juste un jeton de bot)
- [Discord](/fr/channels/discord)
- [Tous les channels](/fr/channels)
