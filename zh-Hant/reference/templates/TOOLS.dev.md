# TOOLS.md - User Tool Notes (editable)

此檔案用於記錄您關於外部工具和慣例的筆記。
它不會定義有哪些工具存在；OpenClaw 在內部提供內建工具。

## 範例

### imsg

- 傳送 iMessage/SMS：描述對象/內容，傳送前確認。
- 偏好簡短訊息；避免傳送秘密資訊。

### sag

- 文字轉語音：指定語音、目標喇叭/房間，以及是否串流。
