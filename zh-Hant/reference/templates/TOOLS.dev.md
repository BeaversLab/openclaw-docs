# TOOLS.md - 使用者工具說明（可編輯）

此檔案供您用來記錄關於外部工具和慣例的筆記。
它不會定義存在哪些工具；OpenClaw 在內部提供內建工具。

## 範例

### imsg

- 傳送 iMessage/SMS：描述對象/內容，傳送前請確認。
- 建議使用簡訊；避免傳送機密資訊。

### sag

- 文字轉語音：指定語音、目標喇叭/房間，以及是否串流傳輸。
