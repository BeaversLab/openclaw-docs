# TOOLS.md - 使用者工具筆記（可編輯）

此檔案用於您關於外部工具與慣例的筆記。
它不定義存在哪些工具；OpenClaw 在內部提供內建工具。

## 範例

### imsg

- 傳送 iMessage/SMS：描述對象/內容，傳送前確認。
- 偏好簡短訊息；避免傳送機密。

### sag

- 文字轉語音：指定語音、目標喇叭/房間，以及是否串流播放。

新增您希望助理了解的任何其他本地工具鏈相關資訊。
