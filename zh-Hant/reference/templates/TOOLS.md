# TOOLS.md - 本地筆記

技能定義了工具是*如何*運作的。這個檔案是用來記錄*你的*具體細節——即你設置中獨特的內容。

## 什麼內容適合放在這裡

例如：

- 攝影機名稱和位置
- SSH 主機和別名
- TTS 的首選語音
- 揚聲器/房間名稱
- 設備暱稱
- 任何特定於環境的內容

## 範例

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 為什麼要分開？

技能是共享的。你的設置屬於你。將它們分開意味著你可以在更新技能時不會遺失你的筆記，並且在分享技能時不會洩露你的基礎設施資訊。
