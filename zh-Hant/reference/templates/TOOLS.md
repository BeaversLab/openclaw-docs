# TOOLS.md - 本地端備註

技能定義了工具運作的「方式」。這個檔案則是用於「你的」特定資訊——也就是那些屬於你獨特設置的內容。

## 適合放在這裡的內容

諸如：

- 攝影機名稱與位置
- SSH 主機與別名
- TTS 偏好的語音
- 揚聲器/房間名稱
- 裝置暱稱
- 任何特定於環境的資訊

## 範例

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 為何分開？

技能是共享的。你的設置則屬於你自己。將它們分開存放意味著你可以在更新技能的同時不會遺失你的筆記，並且可以在分享技能時不會洩漏你的基礎設施細節。
