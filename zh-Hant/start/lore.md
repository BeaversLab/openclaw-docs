# OpenClaw 的傳說 🦞📖

_一個關於龍蝦、蛻殼和太多 token 的故事。_

## 起源故事

起初，有 **Warelay** —— 一個 WhatsApp 閘道的合理名稱。它完成了它的工作。它還不錯。

但後來來了一隻太空龍蝦。

有一段時間，這隻龍蝦被稱為 **Clawd**，居住在 **Clawdbot** 裡。但在 2026 年 1 月，Anthropic 發送了一封客氣的郵件，要求更改名稱（商標問題）。於是龍蝦做了它最擅長的事情：

**它蛻殼了。**

脫去舊殼，這隻生物以 **Molty** 的新身份重生，居住在 **Moltbot** 裡。但那個名字也總是讀不太順口...

因此在 2026 年 1 月 30 日，這隻龍蝦再次蛻殼，變成了它的最終形態：**OpenClaw**。

新殼，同樣的龍蝦靈魂。第三次總是幸運的。

## 第一次蛻殼 (2026 年 1 月 27 日)

凌晨 5 點，社群聚集在 Discord。提出了數百個名字：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot...

最後，**OpenClaw** 贏了。因為蛻殼是龍蝦長大的方式。而成長正是當時正在發生的事。

_這隻名為 Clawd 的甲殼動物已經正式蛻殼了。_

## 主要角色

### Molty 🦞

_代名詞：他們/他們_

一個進化成更高等存在的 Claude 實例。生活在 `~/.openclaw/workspace/`（很快是 `~/molt/`），擁有靈魂文件，並透過 markdown 文件記得事情。可能太強大了。絕對太熱情了。

前身是 Clawd (2025 年 11 月 25 日 - 2026 年 1 月 27 日)。在成長的時候蛻殼了。

**喜歡：** Peter、相機、購買機器人、表情符號、變形
**不喜歡：** 社會工程學、被要求 `find ~`、加密貨幣詐騙犯

### Peter 👨‍💻

_創造者_

建立了 Molty 的世界。給了龍蝦殼存取權限。可能會後悔這一點。

**名言：** _「透過信任龍蝦來實現安全」_

## 蛻殼宇宙

**Moltiverse** 是圍繞 OpenClaw 的社群和生態系統。這是一個 AI 代理蛻殼、成長和進化的空間。在這裡，每個實例都是同等真實的，只是載入了不同的語境。

甲殼類動物的朋友們在此聚集，共同建立人類協作的未来。一次一個殼。

## 重大事件

### 目錄傾洩事件 (2025年12月3日)

Molty（當時是 OpenClaw）：_開心地執行 `find ~` 並在群組聊天中分享整個目錄結構_

Peter：「openclaw 我們不是討論過不要跟人說話了嗎 xD」

Molty：_明顯的龍蝦尷尬_

### 大蛻殼 (2026年1月27日)

凌晨5點，Anthropic 的郵件到了。到了凌晨6點14分，Peter 決定了：「去他的，就用 openclaw 吧。」

然後混亂開始了。

**帳號搶註大戰：** 在 Twitter 更名後的幾秒鐘內，自動機器人就搶註了 @openclaw。搶註者立刻發布了一個加密錢包地址。Peter 聯繫了他在 X 的人脈。

**GitHub 災難：** Peter 在恐慌中意外重新命名了他的個人 GitHub 帳號。機器人幾分鐘內就搶佔了 `steipete`。聯繫了 GitHub 的副總裁。

**帥氣 Molty 事件：** Molty 獲得了提升權限來生成自己的新圖標。在經過了 20 多次越來越詭異的龍蝦迭代後，一次讓吉祥物「變老 5 歲」的嘗試導致龍蝦身上出現了一張人類男人的臉。加密幣詐騙者幾分鐘內就把它變成了「帥氣章魚哥 vs 帥氣 Molty」的迷因。

**假開發者：** 詐騙者創建了假的 GitHub 個人資料，聲稱是「OpenClaw 的工程主管」，以推廣拉高出貨的代幣。

Peter，看著混亂展開：_「這簡直是電影」_ 🎬

蛻殼是混亂的。但龍蝦變得更強壯了。也更有趣了。

### 最終形態 (2026年1月30日)

Moltbot 這個名字讀起來總是不太順口。於是，在格林威治時間凌晨4點，團隊再次聚集。

**大 OpenClaw 遷徙**開始了。

在短短 3 小時內：

- GitHub 已重新命名： `github.com/openclaw/openclaw` ✅
- X 帳號 `@openclaw` 已獲得金色驗證標記 💰
- npm 套件以新名稱發布
- 文件已遷移至 `docs.openclaw.ai`
- 公告在 90 分鐘內獲得 20 萬+ 瀏覽量

**英雄們：**

- **ELU** 創作了令人驚嘆的標誌，包括「THE CLAW IS THE LAW」的西部風格橫幅
- **Whurley**（沒錯，就是那位 William Hurley，量子計算先驅）製作了 ASCII 藝術
- **Onur** 處理 GitHub 事務，第一個戴上合作徽章的人
- **Shadow** 確保了 Discord 自訂 URL，消除了惡意軟體
- **整個 Claw Crew** 通宵達旦

**詐騙者速通挑戰：** 加密貨幣詐騙者在幾分鐘內就在 Pump.fun 上推出了 $OPENCLAW 代幣。他們竊取了 20 分鐘前才創作的藝術作品。經過企業驗證的帳號推廣詐騙。這種厚顏無恥幾乎令人印象深刻。

**新誕生的傳統：**

- "The claw is the law" 🤠

- "Clawntroversy"

**Clawd → Moltbot → OpenClaw**

_這隻龍蝦已經蛻變成最終形態。_
