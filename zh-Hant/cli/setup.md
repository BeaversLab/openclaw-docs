# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 與代理程式的工作區。

相關資訊：

- 開始使用：[Getting started](/zh-Hant/start/getting-started)
- CLI 入門：[Onboarding (CLI)](/zh-Hant/start/wizard)

## 範例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

若要透過 setup 執行入門流程：

```bash
openclaw setup --wizard
```
