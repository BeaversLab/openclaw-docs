# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和 Agent 工作區。

相關連結：

- 開始使用：[Getting started](/zh-Hant/start/getting-started)
- CLI 引導：[Onboarding (CLI)](/zh-Hant/start/wizard)

## 範例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

若要透過 setup 執行引導：

```bash
openclaw setup --wizard
```
