# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和 Agent 工作區。

相關連結：

- 入門指南：[入門指南](/zh-Hant/start/getting-started)
- CLI 入門：[入門 (CLI)](/zh-Hant/start/wizard)

## 範例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

若要透過 setup 執行入門程序：

```bash
openclaw setup --wizard
```
