# `openclaw dns`

廣域網路探索 的 DNS 協助程式 (Tailscale + CoreDNS)。目前專注於 macOS + Homebrew CoreDNS。

相關：

- 閘道探索：[探索 (Discovery)](/zh-Hant/gateway/discovery)
- 廣域網路探索設定：[組態 (Configuration)](/zh-Hant/gateway/configuration)

## 設定

```bash
openclaw dns setup
openclaw dns setup --apply
```
