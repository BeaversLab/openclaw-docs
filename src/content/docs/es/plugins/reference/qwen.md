# Complemento Qwen

Agrega compatibilidad para proveedores de modelos Qwen, Qwen Cloud, Model Studio, DashScope, Qwen Oauth, Qwen Portal y Qwen CLI a OpenClaw.

## Distribución

- Paquete: `@openclaw/qwen-provider`
- Ruta de instalación: incluido en OpenClaw

## Superficie

proveedores: qwen, qwencloud, modelstudio, dashscope, qwen-oauth, qwen-portal, qwen-cli; contratos: mediaUnderstandingProviders, videoGenerationProviders
