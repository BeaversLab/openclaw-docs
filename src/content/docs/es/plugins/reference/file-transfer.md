# Complemento de transferencia de archivos

Recupera, lista y escribe archivos en nodos emparejados mediante comandos de nodo dedicados. Evita la truncación de stdout de bash utilizando base64 sobre node.invoke para binarios de hasta 16 MB.

## Distribución

- Paquete: `@openclaw/file-transfer`
- Ruta de instalación: incluido en OpenClaw
