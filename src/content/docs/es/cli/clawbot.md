# `openclaw clawbot`

Espacio de nombres de alias heredado mantenido para compatibilidad con versiones anteriores.

Alias admitido actualmente:

- `openclaw clawbot qr` (mismo comportamiento que [`openclaw qr`](/en/cli/qr))

## Migración

Prefiera los comandos modernos de nivel superior directamente:

- `openclaw clawbot qr` -> `openclaw qr`
