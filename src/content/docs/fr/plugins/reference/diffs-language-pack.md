# Plugin Diffs Language Pack

Ajoute la coloration syntaxique pour les langues en dehors de l'ensemble par défaut de la visionneuse de différences.

## Distribution

- Package : `@openclaw/diffs-language-pack`
- Route d'installation : npm ; ClawHub : `clawhub:@openclaw/diffs-language-pack`

## Surface

plugin

{/* openclaw-plugin-reference:manual-start */}

## Langues ajoutées

Le plugin de base `diffs` met déjà en surbrillance les langages courants documentés dans [Diffs](/fr/tools/diffs). Installez ce pack linguistique si vous souhaitez la coloration syntaxique pour un ensemble plus large de langages pris en charge par Shiki. Si le pack n'est pas installé, ces fichiers sont toujours rendus en texte brut lisible.

Les exemples incluent Astro, Vue, Svelte, MDX, GraphQL, Terraform/HCL, Nix, Clojure, Elixir, Haskell, OCaml, Scala, Zig, Solidity, Verilog/VHDL, Fortran, MATLAB, LaTeX, Mermaid, Sass/Less/SCSS, Nginx, Apache, CSV, dotenv, INI, et les fichiers diff.

Voir [Shiki languages](https://shiki.style/languages) pour le catalogue des langages et des alias en amont de Shiki.
