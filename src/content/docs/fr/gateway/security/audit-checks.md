`openclaw security audit` émet des résultats structurés indexés par `checkId`. Cette
page est le catalogue de référence pour ces ID. Pour le modèle de menace de haut niveau
et les conseils de durcissement, voir [Sécurité](/fr/gateway/security).

exhaustif) :
