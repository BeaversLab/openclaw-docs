# `openclaw tasks flow`

流命令是 `openclaw tasks` 的子命令，而不是独立的 `flows` 命令。

```bash
openclaw tasks flow list [--json]
openclaw tasks flow show <lookup>
openclaw tasks flow cancel <lookup>
```

有关完整的文档，请参阅 [任务流](/en/automation/taskflow) 和 [tasks CLI 参考](/en/cli/index#tasks)。
