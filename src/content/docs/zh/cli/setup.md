# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和代理工作区。

相关：

- 入门指南：[入门指南](/zh/start/getting-started)
- CLI 新手引导：[新手引导 (CLI)](/zh/start/wizard)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

要通过 setup 运行新手引导：

```bash
openclaw setup --wizard
```
