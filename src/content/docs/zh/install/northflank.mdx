使用一键模板在 Northflank 上部署 OpenClaw 并通过 Web 控制界面访问它。
这是最简单的“无需在服务器上使用终端”的路径：Northflank 为您运行 Gateway(网关)。

## 如何开始

1. 点击 [部署 OpenClaw](https://northflank.com/stacks/deploy-openclaw) 打开模板。
2. 如果您还没有账户，请创建一个 [Northflank 账户](https://app.northflank.com/signup)。
3. 点击 **Deploy OpenClaw now**。
4. 设置所需的环境变量：`OPENCLAW_GATEWAY_TOKEN`（使用一个强随机值）。
5. 点击 **Deploy stack** 以构建并运行 OpenClaw 模板。
6. 等待部署完成，然后点击 **View resources**。
7. 打开 OpenClaw 服务。
8. 在 `/openclaw` 打开公共 OpenClaw URL，并使用配置的共享密钥进行连接。此模板默认使用 `OPENCLAW_GATEWAY_TOKEN`；如果您将其替换为密码认证，请使用该密码。

## 您将获得

- 托管的 OpenClaw Gateway(网关) + 控制界面
- 通过 Northflank 卷 (`/data`) 提供持久化存储，以便 `openclaw.json`、
  每个代理的 `auth-profiles.json`、渠道/提供商状态、会话和
  工作区在重新部署后得以保留

## 连接渠道

使用 `/openclaw` 的控制 UI，或通过 SSH 运行 `openclaw onboard` 查看渠道设置说明：

- [Telegram](/en/channels/telegram)（最快 — 只需机器人令牌）
- [Discord](/en/channels/discord)
- [所有渠道](/en/channels)
