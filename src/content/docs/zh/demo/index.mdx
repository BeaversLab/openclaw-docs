# MDX Component Demos

This section is a standalone regression harness for MDX component rendering.

Use these pages to verify two paths:

- Astro components imported from `src/components/mdx-auto`
- Legacy lowercase tags transformed at runtime in `Head.astro`

## Demo pages

- [Cards](/zh/demo/cards)
- [Steps](/zh/demo/steps)
- [Tabs](/zh/demo/tabs)
- [Accordions](/zh/demo/accordions)
- [Asides](/zh/demo/asides)
- [Layout and framing](/zh/demo/layout)
- [Mixed nesting](/zh/demo/mixed-nesting)
- [Legacy tag enhancement](/zh/demo/legacy-tags)

## What to check

- Markdown inside component bodies stays structured
- Lists remain lists
- Fenced code blocks render as code blocks
- Links and inline code survive slot rendering
- Tabs and accordions keep their interaction behavior
- Legacy lowercase tags do not flatten nested Markdown unexpectedly
