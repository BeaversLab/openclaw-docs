# 文件传输插件

通过专用节点命令在配对节点上获取、列出和写入文件。通过 node.invoke 使用 base64 传输，绕过 bash stdout 截断，支持最大 16 MB 的二进制文件。

## 分发

- 软件包：`@openclaw/file-transfer`
- 安装方式：包含在 OpenClaw 中
