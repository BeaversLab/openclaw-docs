# Qwen 插件

为 OpenClaw 添加了 Qwen、Qwen Cloud、Model Studio、DashScope、Qwen Oauth、Qwen Portal、Qwen CLI 模型提供商的支持。

## 分发

- 包：`@openclaw/qwen-provider`
- 安装方式：包含在 OpenClaw 中

## 暴露

提供商：qwen、qwencloud、modelstudio、dashscope、qwen-oauth、qwen-portal、qwen-cli；合约：mediaUnderstandingProviders、videoGenerationProviders
