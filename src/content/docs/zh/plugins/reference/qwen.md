# Qwen 插件

为 OpenClaw 添加 Qwen、Qwen Cloud、Model Studio、DashScope 模型提供商支持。

## 分发

- 包：`@openclaw/qwen-provider`
- 安装方式：包含在 OpenClaw 中

## 暴露

providers: qwen, qwencloud, modelstudio, dashscope; contracts: mediaUnderstandingProviders, videoGenerationProviders
