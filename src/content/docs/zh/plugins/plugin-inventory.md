# 插件清单

此页面是根据 `extensions/*/package.json`、`openclaw.plugin.json`npm
和根 npm 包 `files` 排除项生成的。使用以下命令重新生成：

```bash
pnpm plugins:inventory:gen
```

## 定义

- **核心 npm 包：** 内置于 npm`openclaw`npm npm 包中，无需单独安装插件即可使用。
- **官方外部包：** OpenClaw 维护的插件，未包含在核心 npm 包中，保留在此官方清单中，并通过 ClawHub 和/或 npm 按需安装。
- **仅源代码检出：** 本地仓库插件，不包含在已发布的 npm 构件中，也不作为可安装包进行宣传。

源代码检出与 npm 安装不同：在 npm`pnpm install` 之后，捆绑的
插件从 `extensions/<id>` 加载，因此本地编辑和包本地工作区
依赖项可用。

## 安装插件

使用 **分发** 列来确定是否需要安装。标有
`included in OpenClaw`Gateway(网关) 的插件已存在于核心包中。官方
外部包需要安装一次，然后重启 Gateway。

例如，Discord 是一个官方外部包：

```bash
openclaw plugins install @openclaw/discord
openclaw gateway restart
openclaw plugins inspect discord --runtime --json
```

在启动切换期间，普通的裸包规范仍然从 npm 安装。
当您需要显式源时，请使用 `clawhub:@openclaw/discord` 或 `npm:@openclaw/discord`。
安装后，按照插件的设置文档（例如 Discord(/en/channels/discord)）添加凭据和渠道配置。
请参阅[管理插件](/zh/plugins/manage-plugins)以了解更新、卸载和发布命令。
