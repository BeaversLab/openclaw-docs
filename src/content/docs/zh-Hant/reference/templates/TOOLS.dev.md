# TOOLS.md - 使用者工具說明 (可編輯)

這個檔案是用來存放*您*對於外部工具和慣例的說明。
它並不定義有哪些工具存在；OpenClaw 在內部提供了內建工具。

## 範例

### imsg

- 傳送 iMessage/SMS：描述對象/內容，傳送前先確認。
- 盡量使用簡短訊息；避免傳送秘密資訊。

### sag

- 文字轉語音：指定聲音、目標揚聲器/房間，以及是否要串流。

新增任何您希望助理了解的關於您的本機工具鏈的其他資訊。
