# `openclaw clawbot`

保留舊版別名命名空間以維持向後相容性。

目前支援的別名：

- `openclaw clawbot qr`（行為與 [`openclaw qr`](/zh-Hant/cli/qr) 相同）

## 遷移

建議直接使用現代化的頂層指令：

- `openclaw clawbot qr` -> `openclaw qr`
