# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和代理程式工作區。

相關連結：

- 入門指南：[入門指南](/en/start/getting-started)
- CLI 引導流程：[引導流程 (CLI)](/en/start/wizard)

## 範例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

若要透過 setup 執行引導流程：

```bash
openclaw setup --wizard
```
