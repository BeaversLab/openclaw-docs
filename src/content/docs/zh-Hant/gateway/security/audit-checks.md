`openclaw security audit` 會發出以 `checkId` 為鍵的結構化發現。本頁面是這些 ID 的參考目錄。如需高階威脅模型和加固指南，請參閱 [安全性](/zh-Hant/gateway/security)。

您最有可能在實際部署中看到的高信號 `checkId` 值（未盡列）：
