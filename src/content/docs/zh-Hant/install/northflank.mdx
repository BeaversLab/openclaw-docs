使用一鍵範本在 Northflank 上部署 OpenClaw 並在瀏覽器中完成設定。
這是最簡單的「伺服器上無需終端機」路徑：Northflank 會為您執行 Gateway，
而您可以透過 `/setup` 網頁精靈來設定所有內容。

## 如何開始

1. 按一下 [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) 以開啟範本。
2. 如果您還沒有帳戶，請建立一個 [Northflank 帳戶](https://app.northflank.com/signup)。
3. 按一下 **Deploy OpenClaw now**。
4. 設定必要的環境變數： `SETUP_PASSWORD`。
5. 按一下 **Deploy stack** 來建置並執行 OpenClaw 範本。
6. 等待部署完成，然後按一下 **View resources**。
7. 開啟 OpenClaw 服務。
8. 開啟公開的 OpenClaw URL 並在 `/setup` 完成設定。
9. 在 `/openclaw` 開啟 Control UI。

## 您會獲得

- 託管的 OpenClaw Gateway + Control UI
- 在 `/setup` 進行網頁設定（無需終端機指令）
- 透過 Northflank Volume (`/data`) 進行持久化儲存，讓設定/憑證/工作區在重新部署後得以保留

## 設定流程

1. 造訪 `https://<your-northflank-domain>/setup` 並輸入您的 `SETUP_PASSWORD`。
2. 選擇一個模型/驗證提供者並貼上您的金鑰。
3. （選用）新增 Telegram/Discord/Slack token。
4. 點擊 **Run setup**。
5. 在 `https://<your-northflank-domain>/openclaw` 開啟 Control UI

如果 Telegram 訊息設為配對模式，網頁設定可以批准配對碼。

## 連結頻道

將您的 Telegram 或 Discord token 貼上到 `/setup` 精靈中。如需設定
指示，請參閱頻道文件：
