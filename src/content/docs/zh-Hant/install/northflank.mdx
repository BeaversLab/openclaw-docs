使用一鍵範本在 Northflank 上部署 OpenClaw，並透過網頁控制 UI 進行存取。
這是最簡單的「無需在伺服器上使用終端機」途徑：Northflank 會為您執行 Gateway。

## 如何開始

1. 點擊 [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) 以開啟範本。
2. 如果您尚未擁有帳戶，請建立一個 [Northflank 帳戶](https://app.northflank.com/signup)。
3. 點擊 **Deploy OpenClaw now**。
4. 設定所需的環境變數： `OPENCLAW_GATEWAY_TOKEN` (請使用強隨機值)。
5. 點擊 **Deploy stack** 以建置並執行 OpenClaw 範本。
6. 等待部署完成，然後點擊 **View resources**。
7. 開啟 OpenClaw 服務。
8. 在 `/openclaw` 開啟公開的 OpenClaw URL，並使用您的 `OPENCLAW_GATEWAY_TOKEN` 進行連線。

## 您將獲得

- 託管的 OpenClaw Gateway + Control UI
- 透過 Northflank Volume (`/data`) 提供的持久儲存空間，讓設定/憑證/工作區在重新部署後得以保留

## 連接頻道

使用位於 `/openclaw` 的 Control UI，或透過 SSH 執行 `openclaw onboard` 以取得頻道設定說明：

- [Telegram](/en/channels/telegram) (最快 — 只需要機器人權杖)
- [Discord](/en/channels/discord)
- [所有頻道](/en/channels)
