# MiniMax 外掛程式

新增對 OpenClaw 的 MiniMax、MiniMax Portal 模型提供者支援。

## 發行

- 套件：`@openclaw/minimax-provider`
- 安裝途徑：包含於 OpenClaw 中

## 介面

提供者：minimax、minimax-portal；合約：imageGenerationProviders、mediaUnderstandingProviders、musicGenerationProviders、speechProviders、videoGenerationProviders、webSearchProviders
