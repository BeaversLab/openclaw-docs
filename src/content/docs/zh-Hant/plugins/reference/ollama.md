# Ollama 外掛程式

為 OpenClaw 新增 Ollama、Ollama Cloud 模型提供者支援。

## 發行

- 套件：`@openclaw/ollama-provider`
- 安裝路徑：包含於 OpenClaw 中

## 介面

提供者: ollama, ollama-cloud; 契約: memoryEmbeddingProviders, webSearchProviders
