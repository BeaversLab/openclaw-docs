# 檔案傳輸外掛程式

透過專用節點指令在配對節點上擷取、列出及寫入檔案。透過在 node.invoke 上使用 base64，繞過 bash stdout 截斷，適用於最大 16 MB 的二進位檔案。

## 發行版本

- 套件： `@openclaw/file-transfer`
- 安裝路徑：包含於 OpenClaw 中
