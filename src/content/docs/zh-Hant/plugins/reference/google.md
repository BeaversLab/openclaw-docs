# Google 外掛程式

為 OpenClaw 新增 Google、Google Gemini CLI、Google Vertex 模型提供者支援。

## 發布

- 套件：`@openclaw/google-plugin`
- 安裝途徑：包含於 OpenClaw

## 介面

提供者：google、google-gemini-cli、google-vertex；合約：imageGenerationProviders、mediaUnderstandingProviders、memoryEmbeddingProviders、musicGenerationProviders、realtimeVoiceProviders、speechProviders、videoGenerationProviders、webSearchProviders
