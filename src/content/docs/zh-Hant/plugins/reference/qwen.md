# Qwen 外掛程式

為 OpenClaw 新增 Qwen、Qwen Cloud、Model Studio、DashScope、Qwen Oauth、Qwen Portal、Qwen CLI 模型提供者的支援。

## 發行

- 套件：`@openclaw/qwen-provider`
- 安裝途徑：包含於 OpenClaw 中

## 介面

提供者：qwen、qwencloud、modelstudio、dashscope、qwen-oauth、qwen-portal、qwen-cli；契約：mediaUnderstandingProviders、videoGenerationProviders
