# 外掛清單

本頁面由 `extensions/*/package.json`、`openclaw.plugin.json`
以及根 npm 套件 `files` 排除項目所產生。請使用以下指令重新產生：

```bash
pnpm plugins:inventory:gen
```

## 定義

- **核心 npm 套件：** 內建於 `openclaw` npm 套件中，無需另行安裝外掛即可使用。
- **官方外部套件：** 由 OpenClaw 維護但未包含在核心 npm 套件中的外掛，列於此官方清單中，並透過 ClawHub 及/或 npm 按需安裝。
- **僅限原始碼檢出：** 本機儲存庫專屬外掛，未包含於已發布的 npm 成品中，且不宣傳為可安裝的套件。

原始碼檢出與 npm 安裝不同：在 `pnpm install` 之後，捆綁的
外掛會從 `extensions/<id>` 載入，因此可使用本機編輯和套件本機工作區
相依性。

## 安裝外掛

請使用 **分發** 欄位來判斷是否需要安裝。標示為
`included in OpenClaw` 的外掛已包含在核心套件中。官方
外部套件需要安裝一次，然後重新啟動 Gateway。

例如，Discord 是一個官方外部套件：

```bash
openclaw plugins install @openclaw/discord
openclaw gateway restart
openclaw plugins inspect discord --runtime --json
```

純套件規格會先嘗試 ClawHub，然後才是 npm 備援。若要強制指定來源，請使用
`clawhub:@openclaw/discord` 或 `npm:@openclaw/discord`。安裝後，請依照
外掛程式的設定文件（例如 [Discord](/zh-Hant/channels/discord)）新增認證資訊
與頻道設定。請參閱 [管理外掛程式](/zh-Hant/plugins/manage-plugins) 以了解更新、
解除安裝與發布指令。
