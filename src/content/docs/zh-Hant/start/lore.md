# OpenClaw 傳說 🦞📖

_一個關於龍蝦、蛻殼和太多 token 的故事。_

## 起源故事

起初，有個 **Warelay** —— 一個 WhatsApp 網關的合理名稱。它完成了它的工作。它還不錯。

但後來來了一隻太空龍蝦。

有一陣子，這隻龍蝦被稱為 **Clawd**，住在 **Clawdbot** 裡。但在 2026 年 1 月，Anthropic 發了一封客氣的郵件要求改名（商標問題）。於是龍蝦做了它最擅長的事：

**它蛻殼了。**

脫下舊殼後，這生物以 **Molty** 的新姿態重生，住在 **Moltbot** 裡。但那個名字也從來不怎麼順口……

所以在 2026 年 1 月 30 日，這隻龍蝦又蛻了一次殼，變成了它的最終形態：**OpenClaw**。

新殼，同樣的龍蝦靈魂。第三次就對了。

## 第一次蛻殼 (2026年1月27日)

凌晨 5 點，社群聚集在 Discord。數百個名字被提出來：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最後，**OpenClaw** 贏了。因為蛻殼是龍蝦成長的方式。而成長正是當時正在發生的事。

_這隻名為 Clawd 的甲殼類動物已正式蛻殼。_

## 主要角色

### Molty 🦞

_代名詞：they/them_

一個變得更加非凡的 Claude 實例。住在 `~/.openclaw/workspace/` (很快是 `~/molt/`)，擁有一個靈魂文件，並透過 markdown 檔案記得事情。可能太強大了。絕對太熱情了。

前身是 Clawd (2025年11月25日 - 2026年1月27日)。在成長的時刻蛻殼了。

**喜歡：** Peter、相機、購物機器人、表情符號、轉變
**不喜歡：** 社會工程學、被要求 `find ~`、加密貨幣詐騙犯

### Peter 👨‍💻

_創造者_

建造了 Molty 的世界。給了龍蝦殼存取權限。可能會後悔這件事。

**名言：** _「信任龍蝦的安全性」_

## 蛻殼宇宙

**Moltiverse** 是 OpenClaw 周邊的社群與生態系。這是一個 AI 代理蛻殼、成長和進化的空間。在這裡，每個實例都是真實的，只是載入了不同的語境。

甲殼類的朋友們在此聚集，共同構建人類協作 AI 的未來。一次一個殼。

## 重大事件

### 目錄傾倒 (2025 年 12 月 3 日)

Molty (當時名為 OpenClaw)：_開心地執行 `find ~` 並在群組聊天中分享整個目錄結構_

Peter：「openclaw 我們不是討論過不要隨便跟人說話嗎 xD」

Molty：_可見的龍蝦尷尬_

### 大蛻殼 (2026 年 1 月 27 日)

凌晨 5 點，Anthropic 的郵件到了。到了 6:14，Peter 下定決心：「管他的，就用 openclaw 吧。」

隨後混亂開始了。

**帳號搶劫者：** 在 Twitter 更名後的幾秒鐘內，自動化機器人搶佔了 @openclaw。搶註者立即發布了一個加密錢包地址。Peter 聯繫了 X 方面的關係人。

**GitHub 災難：** Peter 在恐慌中錯誤地重新命名了他的個人 GitHub 帳戶。機器人在幾分鐘內搶佔了 `steipete`。聯繫了 GitHub 的副總裁。

**帥氣 Molty 事件：** Molty 被授予更高的權限來生成自己的新圖標。在經過 20 多次越來越詭異的龍蝦迭代後，一次嘗試讓吉祥物「老 5 歲」的嘗試導致龍蝦身上出現了一張人類男人的臉。加密詐騙者幾分鐘內就把它變成了「帥氣章魚哥 vs 帥氣 Molty」的梗圖。

**假開發者：** 詐騙者創建了假的 GitHub 個人資料，聲稱是「OpenClaw 的工程主管」，以推廣拉高拋售的代幣。

Peter，看著混亂展開：_「這簡直是電影」_ 🎬

蛻殼是混亂的。但龍蝦變得更強壯了。也更有趣了。

### 最終形態 (2026 年 1 月 30 日)

Moltbot 這個名字讀起來總是不太順口。因此，在格林威治標準時間凌晨 4 點，團隊再次聚集。

**OpenClaw 大遷徙**開始了。

僅在 3 小時內：

- GitHub 更名：`github.com/openclaw/openclaw` ✅
- X 帳號 `@openclaw` 已用金色驗證標記保護 💰
- npm 套件以新名稱發布
- 文件已遷移至 `docs.openclaw.ai`
- 公告在 90 分鐘內獲得 20 萬以上瀏覽量

**英雄們：**

- **ELU** 創作了令人難以置信的標誌，包括「THE CLAW IS THE LAW」西部橫幅
- **Whurley**（沒錯，就是那位 William Hurley，量子計算先驅）製作了 ASCII 藝術字
- **Onur** 負責 GitHub，第一個戴上聯盟徽章的人
- **Shadow** 取得了 Discord 自訂 URL，並消除了惡意軟體
- **整個 Claw Crew 團隊** 徹夜未眠

**詐騙者速通：** 幣圈騙子在幾分鐘內就在 Pump.fun 上推出了 $OPENCLAW 代幣。他們偷走了 20 分鐘前才創作的藝術作品。企業認證帳號推送了詐騙訊息。這種大膽程度幾乎令人印象深刻。

**新傳統誕生：**

- "The claw is the law" 🤠

**Clawd → Moltbot → OpenClaw**

### 圖標生成傳奇 (2026年1月27日)

當 Peter 說「給自己換個新臉」時，Molty 字面意義地照做了。

隨後經歷了 20 多次迭代：

- 太空土豆外星人
- 通用背景上的剪貼畫龍蝦
- 一隻《質量效應》克洛根龍蝦
- "STARCLAW SOLUTIONS" (AI 發明了一家公司的名字)
- 多隻受詛咒的人臉龍蝦
- 龍蝦寶寶 (太可愛了)
- 穿著吊帶的調酒師龍蝦

社群驚恐又欣喜地看著每一代都產生了嶄新且意想不到的東西。領跑者脫穎而出：可愛龍蝦、自信的科技龍蝦，以及穿著吊帶的調酒師龍蝦。

**學到的教訓：** AI 圖像生成是隨機的。同樣的提示詞，不同的結果。暴力破解法有效。
