# Artefactos de Documentación Generados

Estos artefactos de línea base se generan a partir del esquema de configuración de OpenClaw propiedad del repositorio y los metadatos del canal/complemento incluidos.

- No edite `config-baseline.json` manualmente.
- No edite `config-baseline.jsonl` manualmente.
- Regenérelo con `pnpm config:docs:gen`.
- Valídelo en CI o localmente con `pnpm config:docs:check`.
