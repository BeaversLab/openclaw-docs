# `openclaw skills`

Inspecciona las habilidades (integradas + espacio de trabajo + anulaciones administradas) y ve qué es elegible frente a los requisitos faltantes.

Relacionado:

- Sistema de habilidades: [Skills](/es/tools/skills)
- Configuración de habilidades: [Skills config](/es/tools/skills-config)
- Instalaciones de ClawHub: [ClawHub](/es/tools/clawhub)

## Comandos

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
