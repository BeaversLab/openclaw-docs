# `openclaw dns`

Ayudantes de DNS para el descubrimiento de área amplia (Tailscale + CoreDNS). Actualmente centrado en macOS + CoreDNS de Homebrew.

Relacionado:

- Descubrimiento de puerta de enlace: [Discovery](/es/gateway/discovery)
- Configuración de descubrimiento de área amplia: [Configuration](/es/gateway/configuration)

## Configuración

```bash
openclaw dns setup
openclaw dns setup --apply
```
