Despliega OpenClaw en Northflank con una plantilla de un clic y termina la configuración en tu navegador.
Esta es la ruta más fácil “sin terminal en el servidor”: Northflank ejecuta el Gateway por ti,
y configuras todo a través del asistente web `/setup`.

## Cómo empezar

1. Haz clic en [Desplegar OpenClaw](https://northflank.com/stacks/deploy-openclaw) para abrir la plantilla.
2. Crea una [cuenta en Northflank](https://app.northflank.com/signup) si aún no tienes una.
3. Haz clic en **Desplegar OpenClaw ahora**.
4. Establezca la variable de entorno requerida: `SETUP_PASSWORD`.
5. Haz clic en **Desplegar stack** para compilar y ejecutar la plantilla de OpenClaw.
6. Espera a que se complete el despliegue y luego haz clic en **Ver recursos**.
7. Abre el servicio OpenClaw.
8. Abre la URL pública de OpenClaw y completa la configuración en `/setup`.
9. Abre la interfaz de usuario de Control en `/openclaw`.

## Lo que obtienes

- OpenClaw Gateway alojado + Interfaz de usuario de Control
- Configuración web en `/setup` (sin comandos de terminal)
- Almacenamiento persistente a través de Volumen de Northflank (`/data`) para que la configuración/credenciales/espacio de trabajo sobrevivan a los redespliegues

## Flujo de configuración

1. Visita `https://<your-northflank-domain>/setup` e introduce tu `SETUP_PASSWORD`.
2. Elige un modelo/proveedor de autenticación y pega tu clave.
3. (Opcional) Añade tokens de Telegram/Discord/Slack.
4. Haz clic en **Ejecutar configuración**.
5. Abre la Interfaz de usuario de Control en `https://<your-northflank-domain>/openclaw`

Si los MD de Telegram están configurados en emparejamiento, la configuración web puede aprobar el código de emparejamiento.

## Conectar un canal

Pega tu token de Telegram o Discord en el asistente `/setup`. Para ver las instrucciones
de configuración, consulta la documentación del canal:

- [Telegram](/es/channels/telegram) (el más rápido: solo un token de bot)
- [Discord](/es/channels/discord)
- [Todos los canales](/es/channels)
