# `openclaw pairing`

批准或检查 DM 配对请求（适用于支持配对的频道）。

相关内容：

- 配对流程：[Pairing](/zh/start/pairing)

## 命令

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
