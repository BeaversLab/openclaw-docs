# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 与 agent 工作区。

相关：

- Getting started：[入门](/zh/start/getting-started)
- 向导：[入门](/zh/start/onboarding)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过 setup 运行向导：

```bash
openclaw setup --wizard
```
