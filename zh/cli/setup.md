# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和代理工作区。

相关：

- 入门：[入门指南](/zh/en/start/getting-started)
- 向导：[入门](/zh/en/start/onboarding)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

要通过设置运行向导：

```bash
openclaw setup --wizard
```
