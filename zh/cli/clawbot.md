# `openclaw clawbot`

为向后兼容而保留的旧版别名命名空间。

当前支持的别名：

- `openclaw clawbot qr`（与 [`openclaw qr`](/zh/cli/qr) 行为相同）

## 迁移

建议直接使用现代的顶级命令：

- `openclaw clawbot qr` -> `openclaw qr`
