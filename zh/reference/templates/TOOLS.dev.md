# TOOLS.md - 用户工具笔记（可编辑）

此文件用于记录您关于外部工具和约定的笔记。它不定义哪些工具存在；OpenClaw 在内部提供内置工具。

## 示例

### imsg

- 发送 iMessage/SMS：描述对象/内容，发送前确认。
- 优先使用短消息；避免发送机密信息。

### sag

- 文本转语音：指定语音、目标扬声器/房间以及是否流式传输。

添加您希望助手了解的有关本地工具链的任何其他内容。
