# TOOLS.md - 用户工具笔记（可编辑）

此文件用于记录你关于外部工具与约定的 _个人笔记_。
它不定义有哪些工具；OpenClaw 在内部提供内置工具。

## 示例

### imsg

- 发送 iMessage/SMS：说明对象与内容，发送前确认。
- 偏好短消息；避免发送机密。

### sag

- 文本转语音：指定声音、目标音箱/房间，以及是否流式播放。

添加任何你希望助理了解的本地工具链信息。
