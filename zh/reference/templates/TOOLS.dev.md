# TOOLS.md - 用户工具说明（可编辑）

此文件用于记录_你_关于外部工具和惯例的说明。
它不定义存在哪些工具；OpenClaw 在内部提供内置工具。

## 示例

### imsg

- 发送 iMessage/SMS：描述发送对象/内容，发送前进行确认。
- 尽量使用简短消息；避免发送机密信息。

### sag

- 文本转语音：指定语音、目标扬声器/房间，以及是否流式传输。

添加任何其他你希望助手了解的关于你本地工具链的信息。
