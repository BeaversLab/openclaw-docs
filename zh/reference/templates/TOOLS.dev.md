# TOOLS.md - 用户工具笔记（可编辑）

此文件用于记录关于外部工具和约定的个人笔记。
它不定义存在哪些工具；OpenClaw 内部提供内置工具。

## 示例

### imsg

- 发送 iMessage/短信：描述发送对象/内容，发送前需确认。
- 首选短消息；避免发送机密信息。

### sag

- 文本转语音：指定语音、目标扬声器/房间，以及是否流式传输。

添加任何其他您希望助手了解的关于本地工具链的信息。
